알바톡을 소개합니다!

알바톡으로 아르바이트 정보를 카카오톡으로 나에게 전송해보세요!
나만의 알바 정보 리스트가 완성됩니다. 알바톡은 크롬(Chrome) 확장프로그램입니다.
여기(링크)에서 다운로드 받아보세요!

Services

#서비스 소개

“어떻게 하면 여러 사이트의 알바 정보를 한 눈에 볼 수 있을까?”

알바톡은 좀 더 편리한 구직을 꾀하는 질문에서 시작했습니다.
알바몬, 알바천국에서 알바를 찾을 땐 알바톡을 이용해보세요!
해당 알바의 제목, 근무 시간, 급여 등의 정보를 카카오톡 “나에게 보내기”로 전송합니다.
알바톡으로 나만의 알바 정보 리스트를 만들어보세요!
알바톡과 함께하면 여러 사이트를 옮겨 다니는 번거로움이 사라집니다.
알바톡은 크롬(Chrome) 브라우저의 확장 프로그램입니다.

How to Use

#사용 방법

  • 1단계

    Chrome브라우저 설치

    알바톡은 웹 브라우저인 Chrome의 확장 프로그램입니다. 먼저 Chrome 브라우저를 설치해주세요.

  • 2단계

    알바톡 설치

    Chrome 웹 스토어에서 알바톡을 설치해주세요.

  • 3단계

    카카오계정 로그인

    알바톡을 실행한 후,카카오톡 계정으로 로그인 해주세요.

  • 4단계

    서비스 접근 권한 허용

    권한 설정에 주의해주세요!
    알바톡은 카카오톡의 "나에게 보내기" 서비스와
    "공유하기" 서비스를 제공합니다.
    [선택] 카카오톡 메세지 전송 접근 권한과 [선택] 카카오 서비스 내 친구목록 접근 권한을 선택하지 않을 시 위의 서비스가 제한 됨을 알려드립니다.


  • 5단계

    서비스 사용

    이제부터 본격적으로 아르바이트를 찾아봅시다!
    알바천국 혹은 알바몬 사이트에 접속해주세요.

  • 6단계

    실행하기

    마음에 드는 알바를 찾으셨나요?
    확장 프로그램 탭에서 알바톡 로고를 클릭하여
    알바톡을 실행합니다.
    짠!
    해당 알바의 제목, 근무 시간, 급여, 주소는
    자동으로 채워집니다.
    필요한 추가메모를 적을 수도 있습니다.

  • 7단계

    공유하기

    화면 아래의 '내게 보내기' 또는
    '친구에게 공유하기' 버튼을 눌러 공유해보세요!!

  • 8단계

    실행하기

    마음에 드는 알바를 찾으셨나요?
    확장 프로그램 탭에서 알바톡 로고를 클릭하여
    알바톡을 실행합니다.
    짠!
    해당 알바의 제목, 근무 시간, 급여, 주소는
    자동으로 채워집니다.
    필요한 추가메모를 적을 수도 있습니다.
